# Mobile Application Development – Practical 5  
## Jetpack Compose Authentication App with Navigation

This practical demonstrates **Jetpack Compose Navigation** with a complete **Login–Registration flow**, modern UI, reusable components, and clean project structure — following the latest Android development practices.

---

##  Project Overview
This Android application showcases:

-  **User Authentication UI** (Login + Registration)
-  **Jetpack Compose Navigation** for seamless screen transitions
-  **Material 3 UI** with custom form components
-  **State management** using Compose’s `remember` & `mutableStateOf`

---

##  What We Built

###  **Authentication Flow**
- **Login Screen**
  - Email & Password fields
  - "Forgot Password?" link
  - Login button
  - Navigation to Registration
- **Registration Screen**
  - Name, Phone, City input
  - Email, Password, Confirm Password fields
  - Registration button
  - Navigation back to Login

---

##  UI Components
- **Custom Form Fields**
  - Reusable text input components
  - Numeric-only input support
  - Password fields with secure visual transformation
- **Material Design 3**
  - Modern UI theming
  - Elevated cards for form containers
- **Branding**
  - Integrated **Guni Pink** logo in authentication screens

---

##  Technical Implementation

###  Jetpack Compose  
Modern declarative UI framework used for all screens and components.

###  Navigation Compose  
Handled navigation between:
- `login`
- `register`
